Sirius Animator

Gemoc ANR Project #ANR-12-INSE-0011, Program INS

Sirius 2.0

Sirius Animator brings execution capabilities for your DSL or any EMF based models in a quick and flexible way.

Known usages and maturity level

The Sirius Animator has been used in an experimental branch of Arduino Designer to bring instant feedback on a given design without having to compile and deploy the program on the Arduino target.

Plugins integrating UML Designer, an fUML engine and the Eclipse Debug API have also been prototyped.
