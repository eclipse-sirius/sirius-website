4' Tutorial

This tutorial shows the first steps to create a first graphical modeling tool with Sirius.

It relies on a simple DSL and a sample model based on this DSL.

With this video, you will see how to:

  • create a Sirius project (Viewpoint Specification project)
  • initialize a Diagram to graphically vizualize and edit the model elements
  • define Nodes and Edges to display model elements on the diagram
  • add a Creation Tool to create new model elements from a palette

Download the resources used by this video

The DSL and the sample model used by this video can be found here:

  • Family DSL (link to http://www.eclipse.org/forums/index.php/fa/17018/)
  • Sample model (link to http://www.eclipse.org/forums/index.php/fa/17017/)

Please follow the steps below:

  • Install Sirius using instructions from http://www.eclipse.org/sirius/download.html
  • Import the family DSL (File > Import > General > Existing project into workspace). It contains the three projects corresponding to the DSL used in the video.
  • From your Eclipse containg these three plugin projects, launch a Eclipse runtime with sufficient JVM memory, "-XX:MaxPermSize=256m -Xms40m -Xmx512m" for example.
  • In the runtime go to the Sirius perspective and create a new Modeling Project named "basicfamily.sample"
  • Import in this project the sample model used by the video (example.basicfamily).
  • From now you can follow the video to browse the sample model and create your modeler based on basicfamily DSL.
